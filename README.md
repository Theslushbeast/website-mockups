# Website Mockups Demo Pack

Static demo websites for showing potential small-business clients at a business expo.

## Pages

- `index.html` - launcher page for all demos
- `restaurant.html` - restaurant / cafe demo
- `construction.html` - contractor demo
- `retail.html` - retail catalog demo
- `clinic.html` - clinic / dental / wellness demo
- `professional.html` - professional services demo

## GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload every file in this folder to the repository root.
3. Go to repository `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root` folder.
6. Save. GitHub will publish the site at a URL like:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```

The homepage is `index.html`, and each demo page is linked from there.
